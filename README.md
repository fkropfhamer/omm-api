# MEMES API

## INSTALL
required:
- node 
- mongodb 
- yarn


`yarn install`

## START
`yarn start`

## TEST
`yarn test`

## DOCKER 

### START
`docker-compose up`

### START and REBUILD
`docker-compose up --build`
